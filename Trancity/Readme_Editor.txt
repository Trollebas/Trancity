Прочитайте меня! Это руководство по Редактору Trancity (Transedit).

0) Установка и запуск

Установка осуществляется путём распаковки содержимого архива в любое удобное место.
Обращаю внимание: запуск игры напрямую из архива приведёт к непредсказуемым ошибкам!

Минимальные требования для запуска игры:
• Windows XP или выше;
• .NET Framework: 2.0 (только для Windows XP) или 3.5 SP1 (для более новых ОС Windows);
• Последний DirectX 9.0c (Jun 2010);
• SlimDX Runtime для .NET 2.0 (скачать можно здесь: https://code.google.com/archive/p/slimdx/downloads);

1) Основная информация

При запуске у вас будет 3D режим просмотра. Чтобы его сменить на вид сверху (именуется "режим просмотра"), нажмите Ctrl+Tab. 
Редактор способен редактировать карты. В нём можно строить спаины в виде дорог или рельс, 
устанавливать на сплаины светофоры и сигналы, создавать маршруты по сплаинам, парки транспорта.
Так же возможна прокладка как тролелйбусной контактной сети, так и трамвайной. Можно устанавливать объекты. Основные координаты указа-
ны в метрах для удобства редактирования текстовым редактором .city файла.

Управление в редакторе осущевствляется с помощью клавиш и их сочетаний.

2) Основные:
• A/Z - В режиме просмотра поворачивает карту. Пока не работает с остановками, светофорами и сигналами, объектами. Рендер повёрнутых 
сплаинов пока неадекватен. При сохранении карты угол поворота карты не сохраняется и не запоминается при следующем запуске.
• Q - вернуть карту в начальное положение после её поворота.
• Ctrl - При создании кривых сплайнов ведёт сплайн по радиусу угла поворота. При создании контактной сети выравнивает её.
• Ctrl+S - Сохранить карту.
• Ctrl+Tab - Сменить режим просмотра на 3D-режим. Управление поворотом камеры на данный момент неадекватное.
• Ctrl+W - Смена режим просмотра на режим карты. Не предназначен для 3D вида.
• Ctrl+NumPlus и Ctrl+NumMinus в режиме просмотра изменяют масштаб.
• Ctrl+U - Отменить последнее действие. Не работает на изменения в GUI (маршруты, наряды).
• Shift - При создании контактной сети и сплайнов размещает их по специальной сетке 1м.


3) Создание сплайнов (дорог и рельс):
Для создания сплайна потребуется выбрать пункт "Рельсы и дороги". Теперь выбираем дороги или рельсы, при выборе дороги будет высвечи-
ваться соответствующая иконка. Переходим к их созданию.
Для создания прямого участка выбирайте прямую рельсу. Можно создать по сетке 1м по клавише Shift. Кликаем по земле левой кнопкой мыши,
появилось синее удлиняющееся выделение, что и есть наш сплайн. Следить за его длинной можно в значении "l:". После того, как была 
выбрана нужная длина, нажмите левую кнопку мыши ещё раз. Сплайн создался. 
Для присоединения следующего сплайна требуется его подвести к существующему и проделать те же самые действия.
Для создания кривого сплайна, действия почти идентичны, повернуть вторую часть сплайна можно с помощью клавиш A/Z.
Поднимать сплайны на D/C. Расширить или сузить ширину у дорог можно на S/X.
Для того, чтобы ваш поворот был хорошим, можно зажать клавишу Ctrl, позволяющая строить поворот без выпуклостей наружу по
определённому радиусу или делать расширение/сужение дороги прямым.
НЕ ЗАБЫВАЙТЕ СОЕДИНЯТЬ СВОИ СПЛАЙНЫ! Если не соеденить, маршруты просто не проложутся по сплайнам. 
На месте остановок нужно обрезать сплайны, чтобы ПС не останавливался, к примеру, вначале перекрёстка.
Если вам надо выбрать сплайн, но не удалить, нажмите галочку "Режим выбора". Удаляются сплайны левой кнопкой мыши.

4) Создание контактной сети:
Для создания контактной сети выберете пункт "Контактная сеть".
Для создания троллейбусной выберете значёк с текстом "Троллейбусная КС". Для трамвайной выберете цифру 1.
Контактная сеть строится почти аналогично прямым сплайнам, отличия в том, что для ровной прокладки КС нужно зажимать клавишу Ctrl,
а повернуть можно с помощью клавиш A/Z, поднимать D/C. Удалить участок КС можно как левой кнопкой мыши, а если вам попался багганный
троллейбусный, то для этого можно воспользоваться нажатием на колёсико мыши.
Стрелки на троллейбусной контактной сети стройте аккуратно, немного оттянув в бок от цетрального провода. Стрелка не должна быть 
огромной, а так, чтобы пройти и забыть.
Трамвайную КС старайтесь делать либо небольшим зиг-загом, либо прямой. На поворотах особо не отклоняйте КС.
(P.S. Трамвайную КС можно использовать как тросы для РусТролла)

5) Создание парка:
Для создания парков транспорта выберете пункт "Парки". Изначально в новосозданном мире будет парк с названием "Парк". 
Название сменяемо. Чтобы создать условия для отстоя транспорта в парке, нужно выбрать специальные сплайны. Сначала устанавливаем 
въезд ЛКМом по нужному сплайну, сплайн въезда будет зелёного цвета. Затем устанавливаем выезд, будет выделен красным цветом. После, 
можно выбрать "Пути стоянки" выделить те участки, на которых будут отстаиваться транспорт.
Парковые рейсы начинать и заканчивать на выезде и въезде соотственно.

6) Создание остановки:
Для создания остановочных пунктов транспорта выберете пункт "Остановки". Далее устанавливаем остановку на отдельнй сплайн (то есть, не
в притык перекрёсткам и не на огромную чать дороги. Если остановка установлена на рельсы, галочки трогать не надо, если на автолорогу,
то снимаем галочку с Трамвая, галочки на Автобус и Троллейбус поставятся сами. Далее меняем название остановки, если нет идей, то 
просто нумеруем, чтобы не спутать впоследствии. Маршруты на самой остановке прописывать не надо, если их много - можно поменять модель
остановки. Перемещение остановок работоспособно. 
Пропись новых сплайнов на остановке работает неадекватно, новый сплайн, не перемещая остановку, не работает и маршруты его не видят.

7) Создание светофора и 8) сигнала пока описано не будет. • S/X - В режиме постройки светофоров изменяют его положение относительно 
дороги.

9) Создание маршрута:
Для создания маршрута потребуется выбрать пункт "Маршруты" и нажать кнопку "Добавить рейс". Далее ищем конечную маршрута и оттуда
начинаем прокладку маршрута, выеделяя сплайны левой кнопкой мыши. Чтобы не выделять каждый сплайн, можно нажать клавишу Z, позволяющую при прокладке автоматически дойти
до первого перекрёства или разветвления и телепортировать туда камеру. После того, как вы проложили почти весь маршрут, закончите его
на сплаине, откуда начали маршрут.
Парковый рейс отличается тем, что начальный пункт - это выезд из парка. Если была установлена галочка "В парк", то при следовании у ТС
(транспортного средства) будет установлена табличка "В ПАРК" и его конечным пунктом будет вьезд в парк.
В игре есть возможность как автоматического просчёта времени, так и ручного, когда вы сами обкатываете маршрут, но всегда следите 
за тем, что если это трамвайный или троллейбусный маршруты, всегда имели на всём своём протяжении контактную сеть(актуально для
0.6.3.1).

10) Создание наряда
Наряд - расписание движения на отдельном маршруте. Чтобы открыть меню нарядов, нужно выбрать любой маршрут и нажать на галочку 
"Показать наряды". После, если нет наряда, создаём его. Обязательно указываем парк приписки, и обязательно тот, для какого ТС он пред-
назначен. К примеру, прописывать трамвайный парк на троллейбусном или автобусном маршруте глупо и бессмысленно.
Далее добавляем рейсы. Время отправления можно указать любое возможное, от 00:00 до 23:59. Трассу рейса можно выбрать любую существую-
щую на маршрутею Время прибытия будет согласно рассчитано-
му самой игрой или вами прибавленное "Время в пути", указанному в рейсе. Так же возможно ограничить выбор для ботов на маршрут, уста-
новив соответственно любой ТС, присутствующей у вас.

11) Создание объектов
В виде объекта можно загрузить любую модель, будь это какой-либо дом, инфрастуктура контактной сети и много чего ещё. Чтобы открыть
меню объектов, выберете пункт "Объекты". На данный момент в сборке версии доступно 3 дома Building с разными цветами окраски. Выбрав
один из объектов, его можно установить в мир, нажав на кнопку "Добавить объект" и выбрав ему местоположение. Объекту будет присвоен 
номер (Obj x). Повернуть объект можно как клавишами A/Z, так и Ctrl+← и Ctrl+→. Поднять объект - D/C, Ctrl + D/C - подьём на 0,1 метра.
